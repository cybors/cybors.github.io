网易云音乐

在Ubuntu下如何听高品质的音乐，虾米还是网易云，无意间看到了kigawas的网易云音乐命令行版本。 但是在安装过程中遇到很多问题，并不在作者的FAQ里面
遇到的报错，我的Py环境py2和py3里面都没有安装future Module，在pip install future 的环节中反复报错有好多的依赖包没有安装，依次安装后成功安装。

键盘快捷键

J Down 下移
K Up 上移
H Back 后退
L Forword 前进
U Prev page 上一页
D Next page 下一页
F Search 快速搜索
[ Prev song 上一曲
] Next song 下一曲
= Volume + 音量增加

  • Volume - 音量减少
    SpacePlay/Pause 播放/暂停
    ?Shuffle 手气不错
    MMenu 主菜单
    PPresent/History 当前/历史播放列表
    IMusic Info 当前音乐信息
    ⇧+PPlaying Mode 播放模式切换
    AAdd 添加曲目到打碟
    ZDJ list 打碟列表
    SStar 添加到收藏
    CCollection 收藏列表
    RRemove 删除当前条目
    ⇧+JMove Down 向下移动当前项目
    ⇧+KMove Up 向上移动当前项目
    ⇧+CCache 缓存歌曲到本地
    ,Like 喜爱
    .Trash FM 删除 FM
    /Next FM 下一FM
    QQuit 退出
    WQuit&Clear 退出并清除用户信息

Ubuntu安装方法
sudo pip install NetEase-MusicBox
sudo apt-get install mpg123
